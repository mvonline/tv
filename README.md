# Titan TV — Iranian Channel App for Titan OS (Philips TV)

A hosted HTML5 web app for Titan OS / Philips Smart TV that streams Iranian TV channels scraped from parsatv.com. Navigated entirely with a D-pad remote.

---

## Project Structure

```
/
├── index.html          TV app entry point
├── app.js              Main app logic (vanilla JS)
├── style.css           TV-optimized styles
├── channels.json       Generated by scraper
├── requirements.txt    Python dependencies
├── scraper.py          Channel data scraper
├── stream_checker.py   Stream URL validator
└── assets/
    └── logo.svg        App logo (optional)
```

## Prerequisites

### Install uv

[uv](https://docs.astral.sh/uv/) is a fast Python package and project manager. Install it once on your machine:

**macOS / Linux:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**Windows (PowerShell):**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

After installing, restart your terminal so `uv` is on your PATH.

## Part 1 — Python Setup & Scraper

### 1. Create a virtual environment

Inside the project folder, run:

```bash
uv venv
```

This creates a `.venv` folder in the current directory.

### 2. Activate the virtual environment

**macOS / Linux:**
```bash
source .venv/bin/activate
```

**Windows (Command Prompt):**
```cmd
.venv\Scripts\activate.bat
```

**Windows (PowerShell):**
```powershell
.venv\Scripts\Activate.ps1
```

You should see `(.venv)` at the start of your prompt.

### 3. Install dependencies

```bash
uv pip install -r requirements.txt
```

`requirements.txt` contains:
```
requests
beautifulsoup4
```

> **If the site requires JavaScript rendering**, also install Selenium:
> ```bash
> uv pip install selenium
> ```
> Then download [ChromeDriver](https://chromedriver.chromium.org/) matching your Chrome version and place it in your PATH.

### 4. Run the scraper

```bash
python scraper.py
```

This will:
- Fetch the parsatv.com homepage
- Discover all channel links grouped by category
- Visit each channel page and extract the stream URL (`.m3u8` / `.mp4`)
- Save all results to `channels.json`

Output example:
```
2026-03-11 10:00:00 [INFO] Starting parsatv.com scraper...
2026-03-11 10:00:01 [INFO] Found 120 channel links on homepage
2026-03-11 10:00:02 [INFO] [1/120] Processing Manoto TV
2026-03-11 10:00:02 [INFO]   Found stream: https://...manoto.m3u8
...
2026-03-11 10:02:10 [INFO] Done! Scraped 120 channels, 98 with stream URLs.
2026-03-11 10:02:10 [INFO] Results saved to: channels.json
```

### 5. Validate stream URLs

After scraping, run the stream checker to test each URL and mark it working or broken:

```bash
python stream_checker.py
```

This updates `channels.json` with `"working": true` or `"working": false` for every channel. The TV app automatically hides channels where `working` is `false`.

Output example:
```
2026-03-11 10:03:00 [INFO] Total channels: 120, with stream URLs: 98
2026-03-11 10:03:00 [INFO] Checking 98 streams with 5 workers...
2026-03-11 10:03:02 [INFO]   OK [200] Manoto TV
2026-03-11 10:03:03 [WARNING]   FAIL [404] Some Channel
...
2026-03-11 10:03:30 [INFO] Done! Working: 85, Broken: 13, No URL: 22
```

## Part 2 — Running the HTML App

The app is a static web page. Because it fetches `channels.json` via `fetch()`, it **must be served over HTTP** — opening `index.html` directly from the file system will not work.

### Serve with Python (simplest)

Make sure your virtual environment is still active, then run:

```bash
python -m http.server 8080
```

Open your browser and go to:
```
http://localhost:8080
```

### Serve with uv + a one-liner (no install needed)

```bash
uv run --with httpx python -m http.server 8080
```

### Other options

```bash
# Node.js
npx serve .

# Node.js http-server
npx http-server . -p 8080
```

## Part 3 — Hosting for Production

### Deploy to Vercel

1. Install the Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Deploy from the project root:
   ```bash
   vercel
   ```

3. Your app is live at `https://your-project.vercel.app`.

### Deploy to Netlify

**Via CLI:**
```bash
npm i -g netlify-cli
netlify deploy --prod --dir .
```

**Via Git:**
1. Push this folder to a GitHub repo.
2. Go to [netlify.com](https://netlify.com) → **New site from Git**.
3. Set **Publish directory** to `/` (root).
4. Deploy. Your URL: `https://your-site.netlify.app`.

### Deploy to GitHub Pages

1. Push to a GitHub repo.
2. Go to **Settings → Pages → Source**: select the `main` branch, root `/`.
3. Your app is available at `https://username.github.io/repo-name`.

## Part 4 — Submitting to Titan OS Partner Portal

1. Host the app at a stable public HTTPS URL (Vercel or Netlify recommended).
2. Ensure `channels.json` is served from the same origin or a CORS-enabled CDN.
3. Go to the **Titan OS Partner Portal** (contact Philips / TP Vision for access credentials).
4. Submit your app as a **Web App** entry:
   - **App Name:** e.g. `Titan TV`
   - **App URL:** `https://your-project.vercel.app`
   - **Category:** Entertainment / TV & Video
   - **Supported Resolutions:** 1920×1080
5. After review and approval, your app appears in the Titan OS app store.

## Configuration

Edit the `CONFIG` object at the top of `app.js` to customise the app:

```js
const CONFIG = {
  channelsUrl:      './channels.json',  // Path or URL to channels data
  focusColor:       '#e5a00d',          // Gold focus highlight color
  overlayHideDelay: 3000,               // ms before player HUD auto-hides
  columns:          5,                  // Cards per row in the grid
};
```

## Notes

- The app uses **hls.js** (loaded from CDN) for HLS `.m3u8` stream playback on Chromium.
- Channels with `stream_url: null` or `working: false` are automatically hidden.
- The app scales to fit any screen resolution while preserving the 1920×1080 layout.
- No build tools are required — pure HTML/CSS/JS, deployable to any static host.
